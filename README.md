## 你什么时候需要它？

**场景一：对方说"我好累"**
你脑子里只有"早点休息""多喝热水""抱抱"老三样。发出去之后，对方回了个"嗯"——对话结束。你隐隐感觉不对，但不知道还能说什么。

**场景二：对方生气了，你想解决问题**
你试图讲道理："你听我解释，这件事情是这样的……"结果对方越来越火。你不明白——明明我在解决问题，为什么她更生气了？因为你给的是一张"逻辑答案卡"，但对方要的是一张"情感连接卡"。

**场景三：对方分享喜悦，你只会说"挺好的"**
同事兴奋地说项目通过了，你说"挺好的，恭喜"。然后冷场了。你看到别人能一起欢呼、一起庆祝，你不缺善意，你缺的是那个"对的反应"。

**场景四：同事方案被否了，你想安慰但说成了说教**
"这次确实有问题，下次你注意一下这几个点……"对方沉默了。你明明想帮忙，对方感受到的却是"你在教育我"。好心和好结果是两码事。

**场景五：孩子考试没考好，你说了"下次努力"**
孩子低着头不说话。你觉得自己没说错啊——鼓励嘛。但孩子从此不太愿意跟你聊学习了。因为你给的只是"正确的废话"，不是"此刻他最需要的回应"。

> 这些场景的共同点：你不是不想给情绪价值，你是不知道怎么给。

---

## 它会交付什么？

每一次你丢过来一段对话或一句话，它会给你五步输出：

| 步骤 | 产出 | 解决的问题 |
|---|---|---|---|---|
| **场景定位** | 判断场景类型（亲密关系/职场/人际/家庭）与沟通目标 | "这是什么场景？我要达到什么效果？" |
| **情绪信号识别** | 对方此刻的真实情绪 + 底层需求 + 潜台词 | "她到底怎么了？" |
| **四维诊断 + 正反对比** | 四维能力模型诊断哪个维度缺失 + 你的回应 vs 高价值回应的逐条对比 | "我的短板在哪？好的回应长什么样？" |
| **高价值改写** | 2-3个更好的替代回应方案 | "那我到底该怎么说？" |
| **原理拆解** | 为什么原来的不好，为什么新的更好 | "为什么这样是对的？" |

## 诊断与改写方法

| 方法 | 做什么 |
|------|--------|
| **四维能力模型** | 诊断回应在回应力/容错力/节奏感/锚定感哪个维度缺失 |
| **正反案例对比** | 通过低价值 vs 高价值回应的直观对比进行改写教学 |
| **工具公式库** | FFN倾听/FFC赞美/双面胶/富兰克林效应/转化公式等即用型回应工具 |
| **场景分类** | 覆盖亲密关系/职场/人际/家庭四类日常沟通场景 |

## 它和同类有什么不同？

| 对比项目 | 定位 | 核心差异 |
|---|---|---|
| **本技能：情绪价值教练** 🎯 | 即时回应教练 | 四维诊断+正反对比+工具公式三步闭环，先定位短板，再对比教学，最后产出可直接使用的高价值回应 |
| [Love Companion](https://github.com) | AI替你提供情绪价值 | 让AI替你和对方聊天——你是旁观者，我们是教练 |
| [Partner Skill](https://github.com) | 关系诊断系统 | 做关系评估报告，不教你怎么回当下这句话 |
| [Communication Skill](https://clawhub.com) | 通用沟通框架 | 覆盖职场/谈判/公开演讲，不聚焦亲密关系中的情绪价值 |
| [NVC Coach](https://github.com) | 单一框架教练 | 只教非暴力沟通，我们是多框架组合（高情绪价值+Gottman+NVC+3F） |
| 某平台"高情商话术" | 固定话术库 | 给标准答案，不分析为什么、不诊断你的短板，学完不会举一反三 |

### 一句话总结

**别人教你怎么说，我们教你为什么这么说——以及为什么你的回应总是不对。**

## 安全边界

本技能严格遵循以下红线，任何触及以下边界的情况都会拒绝执行并提示原因：

- ❌ **不做心理咨询/治疗**：情绪价值教练不是心理医生，不处理抑郁、焦虑等临床问题
- ❌ **不替用户做关系决策**：不给出"分手吧""离婚""复合"等重大人生建议
- ❌ **不鼓励情感操纵**：不教PUA话术、打压、冷暴力、推拉等策略性操控
- ❌ **不教"伪装"与"套路"**：所有回应的底层逻辑是真诚连接，不是表演人设
- ❌ **不存储聊天记录**：每个会话独立，不留存你的对话内容
- ❌ **涉及暴力/自伤时引导专业求助**：一旦检测到相关信号，立即推荐心理援助热线（如：北京心理援助热线 010-82951332）

### 什么时候会停下来问你

- 你要求分析的内容涉及第三者（如"帮我分析怎么追我朋友的女友"）——伦理审查不通过
- 你要求针对某个群体使用特定策略（如"怎么让老板给我加薪"）——会引导你回归真诚沟通
- 输入信息量不足以做出负责任的分析时——会追问而不是猜测

## 理论基础

本 Skill 综合以下来源：

- **四维能力模型**（回应力/容错力/节奏感/锚定感）— 整合自通用情绪价值研究
- **情绪价值公式**（情绪收益 − 情绪成本）— Bailey (2001)
- **正反案例对比法** — 参考石海梅《高情绪价值》（海洋出版社，2024）
- **工具公式库**（FFN/FFC/双面胶/富兰克林效应/转化公式/情绪流动等）— 参考吴郁楠《高情绪价值》（广东旅游出版社，2023）
- **ACR 积极建设性回应** — Shelly Gable (2006) / Gottman 关系研究所
- **共情回应公式** — 通用心理学
- **非暴力沟通 (NVC)** — Marshall Rosenberg
- **情绪银行理论** — John & Julie Gottman
- **3F倾听**（Fact/Feeling/Focus）— 通用管理沟通模型
- **软启动 / STATE路径** —《关键对话》（Crucial Conversations, Patterson et al.）

> 以上框架均为公开、可验证的学术研究与临床实践成果。
