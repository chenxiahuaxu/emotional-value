### 工具详表

#### FFC赞美法

| 维度 | 内容 |
|------|------|
| **来源确认** | ✅ 吴郁楠《高情绪价值》第1章 |
| **公式** | 感受(Feeling) + 事实(Fact) + 对比(Compare) |
| **何时使用** | 需要真诚赞美某人，但只说"你真棒"太单薄时 |
| **分步说明** | **F-感受**：先说出你的真实感受（"我好佩服你……"）<br>**F-事实**：描述具体行为或事实（"你每天坚持5点起床写作"）<br>**C-对比**：和周围人或过去的自己对比（"一般人都做不到"） |
| **书中示例** | 吴郁楠在书中示例如："我好佩服你（感受），你每天坚持5点起床写作（事实），一般人根本做不到（对比）。" |
| **补充说明** | 对比对象可以是他人、过去的自己，也可以是普遍标准。注意对比要真诚，不要贬低第三方。 |

#### 负面情绪转化公式

| 维度 | 内容 |
|------|------|
| **来源确认** | ✅ 吴郁楠《高情绪价值》第2章 |
| **公式** | 认同情绪 + 换角度提问 + 对比式肯定 |
| **何时使用** | 对方处于负面情绪（生气、委屈、沮丧）中，需要引导其转向积极 |
| **分步说明** | **① 认同情绪**：先接纳对方的感受，不否定（"我知道你现在很难过"）<br>**② 换角度提问**：换个视角提问，引导对方看到不同可能性（"如果换个角度看这件事呢？"）<br>**③ 对比式肯定**：通过对比让对方看到自己的进步或优势（"相比上次，你已经进步很多了"） |
| **补充说明** | 第①步最关键——跳过认同直接给建议会让对方觉得被否定。 |

#### 愤怒疗法七步法

| 维度 | 内容 |
|------|------|
| **来源确认** | ✅ 吴郁楠《高情绪价值》第2章 |
| **步骤** | ① 保持冷静，不激化<br>② 给对方表达空间（不打断）<br>③ 共情回应（"我理解你为什么这么生气"）<br>④ 帮对方厘清真正需求<br>⑤ 引导对方看到积极面<br>⑥ 共同寻找解决方案<br>⑦ 确认情绪缓解并感谢对方的信任 |
| **何时使用** | 对方怒气冲冲，普通安慰失效时 |
| **补充说明** | 书中强调前两步最容易失败——对方发怒时，本能反应是解释或反击，但恰好需要做的是闭嘴和倾听。 |

#### 万能回应公式（放大高兴）

| 维度 | 内容 |
|------|------|
| **来源确认** | ✅ 吴郁楠《高情绪价值》第3章 |
| **公式** | 认同 + 追问 + 一起高兴 |
| **何时使用** | 对方分享开心、自豪、成就的事情时 |
| **分步说明** | **① 认同**：先承认这件事确实值得高兴（"太棒了！"）<br>**② 追问**：深入追问细节，表示你真的感兴趣（"你是怎么做到的？"）<br>**③ 一起高兴**：把对方的开心变成共同的开心（"我真为你高兴，今晚必须庆祝一下！"） |
| **书中示例** | 对方说"我升职了"——❌ 低情绪价值回应："哦，那挺好。" ✅ 高情绪价值回应："太棒了！我就知道你一定行（认同）！你准备了多久的述职（追问）？今晚咱们去吃顿好的庆祝一下（一起高兴）！" |
| **补充说明** | 这一步是积极的"情绪放大"——不是简单地附和，而是通过追问让对方的成就感被"看见"。 |

#### FFN倾听模型

| 维度 | 内容 |
|------|------|
| **来源确认** | ✅ 吴郁楠《高情绪价值》第4章 |
| **公式** | Fact（事实）→ Feeling（情绪）→ Need（需求） |
| **何时使用** | 任何需要深度倾听的场景，尤其是对方表达不满、抱怨或复杂感受时 |
| **分步说明** | **F-事实**：对方说了什么？发生了什么？（客观信息）<br>**F-情绪**：对方此刻是什么感受？（生气？委屈？失望？）<br>**N-需求**：对方真正需要什么？（被理解？被支持？解决问题？） |
| **书中示例** | 伴侣抱怨"你最近总是很晚回家"——❌ 低情绪价值回应："我工作忙啊。" ✅ FFN回应：事实(F) = 最近回家确实晚了→ 情绪(F) = 她感到被忽视和孤独→ 需求(N) = 需要更多的陪伴和安全感 |
| **补充说明** | FFN模型可以帮助避免"你以为你听懂了"的误区。回应时将N（需求）反馈给对方，对方会感到被深度理解。 |

#### 情绪流动公式

| 维度 | 内容 |
|------|------|
| **来源确认** | ✅ 吴郁楠《高情绪价值》第5章 |
| **公式** | 表达 → 接收 → 反馈（循环流动） |
| **何时使用** | 亲密关系中，当情绪停滞（一方不愿说、另一方不愿听）时 |
| **补充说明** | 书中指出：情绪不流动的关系会"凝固"。高质量亲密关系的特征不是没有争吵，而是情绪始终在两人之间流动。 |

#### 双面胶赞美法

| 维度 | 内容 |
|------|------|
| **来源确认** | ✅ 吴郁楠《高情绪价值》第5章 |
| **公式** | 认同/赞美 + 建议/希望 |
| **何时使用** | 需要提出建议或批评时，对方不会感到被攻击 |
| **书中示例** | "你做的饭真好吃（认同/赞美），要是下次少放一点盐就更完美了（建议/希望）。" |
| **补充说明** | 和"三明治沟通法"（好-坏-好）类似，但更强调认同/赞美要具体真诚，否则对方会觉得是"套路"。 |

#### 富兰克林效应

| 维度 | 内容 |
|------|------|
| **来源确认** | ✅ 吴郁楠《高情绪价值》第5章 |
| **公式** | 请对方帮一个小忙 → 对方因"帮助行为"而产生好感 |
| **何时使用** | 想快速拉近和某人（同事、新朋友、心仪对象）的距离时 |
| **补充说明** | 书中引用心理学经典效应：人会更喜欢自己帮助过的人。关键在于"小忙"——太小没感觉，太大让对方为难。例如借一本书、请对方帮一个小建议。 |

## 四、与石海梅版的区别

| 对比维度 | 吴郁楠版 | 石海梅版 |
|----------|---------|---------|
| **出版信息** | 广东旅游出版社，2023年12月 | 海洋出版社，2024年8月 |
| **核心框架** | 按问题类型组织（5个"为什么"） | 按关系类型组织（5篇：女性/男性/职场/人际/家庭） |
| **方法论** | 提供具体工具和公式（FFN、FFC、愤怒七步法、万能回应公式等） | 正反案例对比 + 场景话术 |
| **定位** | 为什么 + 怎么办 —— 先诊断问题、再给工具 | 什么场景 + 怎么说 —— 直接给话术模板 |
| **内容侧重** | 偏诊断和底层逻辑，教读者"为什么要这么做" | 偏实操话术，教读者"具体怎么说" |
| **目标读者** | 已经意识到问题但不知道怎么改的人 | 需要"拿来就用"的场景话术的人 |
| **适合阅读方式** | 系统阅读（逻辑链条完整） | 可按场景跳跃阅读（即查即用） |

## 五、两本书在本 Skill 中的配合使用

| 使用层次 | 吴郁楠版 | 石海梅版 |
|---------|---------|---------|
| **诊断层** | FFN倾听模型、负面情绪转化公式——定位问题出在哪里 | 正反案例对比——看看别人在这个场景怎么做 |
| **工具层** | FFC赞美法、万能回应公式、双面胶赞美法——掌握回应的底层公式 | 场景话术库——直接参考特定场景的说法 |
| **练习层** | 书中附录有赞美练习和参考答案 | 每个场景都有"正面示范"和"反面教材" |

**完整学习路径**：
1. 用吴郁楠的 **FFN倾听模型** 先判断对方此刻的事实+情绪+需求
2. 从石海梅的场景话术中找对应的正面示范
3. 用吴郁楠的公式工具（FFC/万能回应/双面胶）构建自己的回答
4. 参照石海梅的反面教材，避免踩雷

> 一句话概括：**吴郁楠给公式和诊断，石海梅给场景和话术；先诊断，后套公式，再参考话术。**
