# 四维能力模型 —— 情绪价值核心诊断框架

> **来源说明**：本章的四维能力模型（回应力/容错力/节奏感/锚定感）整合自通用情绪价值研究、积极倾听理论、Gottman 关系研究等学术来源，作为本Skill的核心诊断框架。该模型**不来自**《高情绪价值》石海梅一书——该书以五篇章场景+正反案例对比为主要方法论。

**一句话定位：** 四维模型把"回应不够好"这个模糊感受，拆解成四个可测量、可练习的能力维度——让你知道具体哪里不行，以及往哪个方向练。

---

## 一、为什么需要四维诊断

在提供情绪价值时，我们常常陷入这样的困境：

- 你知道自己的回应"不太好"，但说不清是哪里不好。
- 你知道对方"不舒服"，但找不到问题的根源。
- 你尝试去改，但不知道具体该往哪个方向改。

**问题出在哪里？**

"情绪回应能力"太笼统了——就像说"我身体不好"一样，你没法直接对这个结论做任何事。你需要体检报告，告诉你"是血压高还是血糖高还是血脂高"。

四维模型就是你的情绪回应"体检报告"。它把一个模糊的"不够好"拆解为四个独立的维度：

| 维度 | 回答的问题 | 如果你缺这个 |
|------|-----------|------------|
| **回应力** | 你接住对方的信号了吗？ | 对方觉得"你没在听""你不理解我" |
| **容错力** | 你能容纳对方的负面情绪吗？ | 对方不敢在你面前暴露脆弱 |
| **节奏感** | 你的回应速度和深度合适吗？ | 对方觉得"和你聊天好累" |
| **锚定感** | 对方能把你当作情感安全基地吗？ | 对方觉得"你靠不住""忽冷忽热" |

> **为什么是这四个维度？**
>
> 回应力解决"听没听到"的问题，容错力解决"接不接得住"的问题，节奏感解决"合不合适"的问题，锚定感解决"靠不靠谱"的问题。四者共同覆盖了情绪回应的完整链路：**接收 → 容纳 → 调整 → 稳定**。

## 二、四维能力模型详解

### 2.1 回应力（Responsiveness）

**定义：** 能否准确捕捉对方的情绪信号并给出让对方感到"你听到了"的反馈。

**核心问题：** 对方的情绪信号，你接收到了吗？你让对方感觉到"你接收到了"吗？

**1-5 分评分标准：**

| 分数 | 行为描述 |
|------|---------|
| **1 分** | 完全忽略对方的情绪信号。对方在表达情绪，你自顾自说别的、看手机、打断对方、转移话题。 |
| **2 分** | 只回应事实层面，忽略情绪层。对方说"我今天被领导批了"，你回"领导为什么批你"——你没接住委屈，直奔解决方案。 |
| **3 分** | 能识别明显的情绪，但回应公式化。"别难过""没事的"——方向对了，但不够具体，像自动回复。 |
| **4 分** | 能准确命名情绪并用具体细节回应。"你刚才说那句话的时候声音有点抖——这件事让你很委屈对吗？"——对方感到被认真倾听。 |
| **5 分** | 精准捕捉情绪信号，且能回应暗含的需求。不仅听到了表面情绪，还听出了对方没说出来的需要——"你不是需要方案，你是需要有人肯定你的付出。" |

**典型信号（什么时候缺失）：**

| 信号 | 表现 |
|------|------|
| 🚩 对方说"算了，不说了" | 你的回应没有给对方继续表达的欲望 |
| 🚩 对方重复说同一件事 | 你的首次回应没有让对方感到"被听到" |
| 🚩 对话变成"你问我答" | 你没在接收，你在审问 |
| 🚩 对方说"你根本不懂" | 情绪信号接收失败 |

**训练方法：FFN 三要素刻意练习**

```
FFN 刻意练习步骤：
1. 对方说完后，在脑子里过一遍：
   F（Fact）→ 刚才他说了什么具体事情？
   F（Feeling）→ 他此刻的情绪是什么？
   N（Need）→ 他真正需要的是什么？
2. 确认 N 之前，不要开口给回应。
3. 每天找一个真实对话，做完 FFN 记录。
```

> **训练指标：** 连续 7 天，每天至少一次在对话中使用 FFN 结构后再开口。7 天后评估：你回应的"被忽略感"是否下降。

**正例 vs 反例：**

| 场景 | ❌ 低回应力 | ✅ 高回应力 |
|------|-----------|-----------|
| 对方说"今天开会我又被怼了" | "为什么被怼？你说了什么？"（直奔事实，忽略情绪） | "又被怼了？听起来你今天整个人都不好了——想跟我说说细节，还是就想吐槽一下？"（接住情绪+给选择权） |
| 对方说"最近好累" | "那就早点睡。"（敷衍式回应） | "你最近真的太忙了——是身体累还是心里累？跟我说说，哪件事最消耗你？"（具体化+关心感受） |
| 对方说"算了，不重要" | "哦，那好吧。"（放弃接球） | "你说算了的时候，语气不太对——这件事其实是重要的，只是你现在不想说了对吗？那等你愿意说了我随时在。"（不强迫但保持连接） |

### 2.2 容错力（Tolerance）

**定义：** 面对对方负面情绪时不防御、不反击、能接纳的能力。

**核心问题：** 当对方的情绪指向你，或者对方情绪激烈时，你能"接住"而不"反弹"吗？

**1-5 分评分标准：**

| 分数 | 行为描述 |
|------|---------|
| **1 分** | 立刻反击或冷战。对方一有负面情绪——尤其是针对你时——你马上防御："那是你的问题""你凭什么这么说我"，或直接沉默冷处理。 |
| **2 分** | 回避或转移。对方表达负面情绪时，你感到不适，于是用"好了好了别说了""我们去吃点东西吧"来回避。你没有反击，但也没有容纳。 |
| **3 分** | 能克制不反击，但内心在忍耐。"我忍住了没说回去"——表面平静，但对方能感受到你的压抑和不自在。 |
| **4 分** | 能接纳对方的负面情绪，不认为对方在攻击自己。"你在生气——这是你的感受，我可以接受。"——不防御、不解释、不反击。 |
| **5 分** | 不仅接纳，还能在对方情绪最激烈时保持连接。"你现在很生气，说的话可能不是你真的想说的——但我还在这里，没有走开。"——对方在最糟糕的状态下依然感到安全。 |

**典型信号（什么时候缺失）：**

| 信号 | 表现 |
|------|------|
| 🚩 对方一生气/指责，你立刻辩解 | "这不是我的问题""你听我解释"——你的优先级的"自证清白"而非"接住对方" |
| 🚩 对方表达负面情绪时你沉默不语 | 你以为你在"冷静"，实际上你在用冷暴力应对 |
| 🚩 你经常说"你别激动""你冷静点" | 你无法容纳对方的激动，所以想让它快点消失 |
| 🚩 对话中你频繁出现"但是""可是""不过" | 你无法真正接纳对方，总想纠正对方 |

**训练方法：响应延迟练习**

```
3 秒窗口训练：
1. 对方说出让你感到不舒服/被针对/想反击的话。
2. 在心里默数 3 秒：1...2...3...
3. 3 秒内只做两件事：深呼吸 + 告诉自己"这不是攻击，是对方的情绪表达"。
4. 3 秒后再开口。

进阶：从 3 秒延长到 10 秒。
前 3 秒：控制本能反应。
中间 3 秒：理解对方的情绪来源。
最后 4 秒：组织回应的内容。
```

> **训练指标：** 连续 14 天，每次想反击时执行 3 秒窗口。14 天后评估：自动反击的次数是否下降了 50% 以上。

**正例 vs 反例：**

| 场景 | ❌ 低容错力 | ✅ 高容错力 |
|------|-----------|-----------|
| 对方说"你从来都不在乎我的感受！" | "我怎么不在乎了？我昨天还……你讲不讲道理？"（立刻反击+自证） | 3 秒深呼吸后："你这么说，肯定是有什么事让你觉得被忽视了——能告诉我是什么事吗？"（不防御，探寻根源） |
| 对方情绪崩溃大喊大叫 | "你冷静点行不行？喊有什么用？"（要求对方收住情绪） | "你现在很难受，我知道。等你缓一缓，我们再聊。"（先接纳，不要求对方改变状态） |
| 对方说"都是你的错" | "凭什么说是我的错？你难道没有问题吗？"（反弹+归因于对方） | "你觉得是我的错——那告诉我，我做了什么让你这么难受？我想听。"（不防御，用提问代替辩解） |

### 2.3 节奏感（Pacing）

**定义：** 把握对话速度和深度的能力——何时追问、何时沉默、何时转话题。

**核心问题：** 你的回应速度和深度，和对方当前的状态匹配吗？

**1-5 分评分标准：**

| 分数 | 行为描述 |
|------|---------|
| **1 分** | 完全无视对方状态，强行推进自己的节奏。对方不想聊了还在追问；对方还想说你就打断转话题。独白式沟通。 |
| **2 分** | 能跟随明显的节奏信号，但经常"慢半拍"或"抢拍"。对方沉默了你还不停追问；对方情绪没平复你就开始说笑。 |
| **3 分** | 基本能跟上对方的节奏，但在关键节点容易出错——比如对方需要深度时你太过轻浮，对方需要轻松时你太过沉重。 |
| **4 分** | 能敏锐感知对方的能量状态并主动调整——对方低落时放慢节奏，对方兴奋时跟上热情。节奏匹配度 > 80%。 |
| **5 分** | 不仅能跟随，还能引导节奏。对方情绪混乱时，你能用节奏把对话带到一个安全、舒适的方向。和你的对话像跳双人舞——自然流畅，双方都舒服。 |

**典型信号（什么时候缺失）：**

| 信号 | 表现 |
|------|------|
| 🚩 对方频繁看手机/看时间 | 你的节奏太慢或太无聊，对方想结束 |
| 🚩 对方回答越来越短（嗯/哦/好） | 你的节奏没有跟上对方的状态——或者推进太快（让ta不想说），或者太慢（让ta不耐烦） |
| 🚩 对方说"先不说这个了" | 你的话题深度超越了对方当前愿意聊的程度 |
| 🚩 对话中有明显的尴尬停顿 | 你没有给对方留出回应空间，或你没有在需要停留的地方停留 |

**训练方法：能量感知练习**

```
能量感知三步法：
1. 观察（10 秒）
   对方的语音语调、语速、肢体语言、眼神 —— 判断当前能量等级：
   - 🔴 高能量（兴奋/激动/愤怒）：语速快、音量大、手势多
   - 🟡 中能量（平静/专注）：匀速、适中音量、有眼神交流
   - 🟢 低能量（疲惫/低落/悲伤）：语速慢、音量小、眼神回避

2. 匹配（下一个回应）
   用和对方匹配的节奏回应：
   - 对方高能量 → 语速稍快，音量匹配，不要压对方
   - 对方中能量 → 正常节奏，保持交流感
   - 对方低能量 → 放慢语速，降低音量，留下沉默空间

3. 引导（确定对方跟上后）
   当你确认节奏匹配后，可以尝试小幅调整节奏（放慢或加快），
   观察对方是否跟随——如果能跟随，说明你们已经在同一个频道上了。
```

> **训练指标：** 每天 3 次有意识地做"能量感知三步法"，记录对方的节奏类型和你匹配后的反应。14 天后，你应该能直觉地感知对方的能量状态而不需要刻意"三步"。

**正例 vs 反例：**

| 场景 | ❌ 低节奏感 | ✅ 高节奏感 |
|------|-----------|-----------|
| 对方刚被批评，情绪低落不想说话 | "怎么不说话了？你说说领导是怎么说的？我帮你分析一下。"（在对方低能量时强推高能量对话） | 安静坐一会儿，然后轻声说："不想说就不说。我陪你坐坐。"（用低能量节奏匹配对方） |
| 对方在兴奋地分享旅行经历 | "嗯，挺好。"（冷冷淡淡，能量完全不匹配） | "天哪！快说说那个地方！你拍照了吗？我最想去那里了！"（用高能量节奏跟上对方） |
| 对方刚说了一件沉重的事，需要消化 | "哈哈哈哈那后来呢？"（对方需要停顿，你快速转场） | 沉默 3-5 秒，然后说："这件事……让我缓一下。你先说说你现在什么感觉？"（留出消化时间） |

### 2.4 锚定感（Anchoring）

**定义：** 让对方在关系中感受到稳定、可靠、一致的情绪支撑。

**核心问题：** 对方能否把你当作"无论发生什么，ta 都会在"的情感安全基地？

**1-5 分评分标准：**

| 分数 | 行为描述 |
|------|---------|
| **1 分** | 忽冷忽热，情绪波动极大。今天热情明天冷漠，对方永远摸不准你的状态——"他今天是开心还是不开心？会不会突然不理我？" |
| **2 分** | 偶尔靠谱但不稳定。大部分时间还行，但关键时候掉链子——对方最需要你的时候你不在或不回应。 |
| **3 分** | 基本可靠但缺乏深度。你能做到"回应"，但回应停留在表面——对方觉得你是一个"还不错但靠不住深处"的人。 |
| **4 分** | 稳定可靠，对方知道你会在。你有一个稳定的回应节奏，对方不需要猜测你的状态。遇到困难时，对方第一个想到的是你。 |
| **5 分** | 情感安全基地。对方在最脆弱时毫不犹豫地找你，因为 ta 知道：①你会认真听 ②你不会评判 ③你不会消失 ④你不会用 ta 的脆弱对付 ta。你是 ta 情绪世界里最确定的存在。 |

**典型信号（什么时候缺失）：**

| 信号 | 表现 |
|------|------|
| 🚩 对方说"你今天怎么这么好" | 你平时的状态太不稳定，偶尔好一次被对方当作"反常" |
| 🚩 对方很少主动找你倾诉 | 你的不稳定让对方养成了"不依赖你"的习惯 |
| 🚩 对方说"我不想打扰你" | 你让ta觉得你的时间是"可以被占用"的，而不是"属于这段关系"的 |
| 🚩 对方需要确认你的状态再说话 | "你现在忙吗？""你在生气吗？"——对方在试探你的状态是否安全 |

**训练方法：一致性与可靠性练习**

```
一致性练习（3 周计划）：

第 1 周：回应时间一致性
- 每天固定时间段回复对方消息（如：早上 9 点前回复昨晚的消息）
- 让对方逐渐建立"ta 会在那个时间段回应"的预期
- ⚠️ 重点不是"秒回"，是"可预期"

第 2 周：回应质量一致性
- 无论你今天心情如何，对对方保持同样的回应质量
- 状态不好时用："我状态不太好，但我还是想听你说——你说，我听着。"
- ⚠️ 不要因为自己情绪不好就冷落对方

第 3 周：情绪稳定性
- 对方表达任何情绪时，你的反应是稳定的（不因话题不同而忽冷忽热）
- 对方可以对你表达任何情绪而不担心你的反应
- ⚠️ 你的任务是"成为对方情绪风暴中的锚"——不是没有风浪，而是你在风浪中不动
```

> **训练指标：** 连续 21 天执行一致性练习。21 天后，观察对方是否更主动地找你分享——主动分享的频率提升了多少？

**正例 vs 反例：**

| 场景 | ❌ 低锚定感 | ✅ 高锚定感 |
|------|-----------|-----------|
| 对方连续几天情绪不好 | 第一天耐心，第二天变冷淡，第三天直接不回。"ta 老是负能量，我不想听了。"（情绪不稳定+忽冷忽热） | 每天保持同样的回应质量。"你最近状态不好——没关系，我每天都会在，你想说就说，不想说就不说。"（稳定的态度+一致的质量） |
| 对方和你分享脆弱 | 今天听了，明天在吵架时提起对方的脆弱："你上次不是说自己很脆弱吗？怎么现在这么强势？"（用脆弱攻击对方——锚定感归零） | 倾听脆弱后，永远不以此作为日后争论的武器。"你上次说的那个事，我一直记着——你后来怎么样了？"（用脆弱加深连接） |
| 对方说"你现在有空吗" | "等一下。""现在忙着。""晚点说。"——从不给确定时间，让对方悬着。 | "现在可能不太方便，我大概半小时后忙完，忙完马上找你。"（给出明确预期） |

## 三、四维速查表

快速对照你的回应习惯，定位哪个维度需要优先练习。

| 你的回应表现 | 对应维度缺失 | 一句话调整方向 |
|------------|------------|---------------|
| 总是直接给方案 | **回应力** | 先问"你感觉怎么样"——跳过情绪直奔方案 = 你没有听到对方 |
| 被对方的情绪触发就反击 | **容错力** | 延迟 3 秒再反应——你的本能反应是防御，不是连接 |
| 对方不想聊了还在追问 | **节奏感** | 观察并跟随对方的能量——对方沉默 = 信号，不是需要你去填充的空白 |
| 时冷时热让对方不安 | **锚定感** | 保持回应的一致性——你的稳定是对方的安全感来源 |
| 对方说事你说理 | **回应力 + 节奏感** | 先确认情绪再做回应——你太快进入"理性分析"模式了 |
| 对方哭了你不知所措 | **容错力 + 锚定感** | 允许对方哭 + 陪在身边——不需要解决问题，只需要在场 |
| 聊天经常冷场 | **节奏感 + 回应力** | 留出沉默空间 + 用具体细节接话——你太紧张节奏，反而不自然 |
| 对方说"你不懂我" | **回应力 + 锚定感** | 用 FFN 重新倾听 + 保持一致的关注——不是一次没做好，是长期积累的印象 |
| 吵架总是翻旧账 | **容错力 + 锚定感** | 只处理当下的事 + 不累积情绪——旧账是你之前没有处理好就放下的 |
| 对方不再主动找你倾诉 | **回应力 + 容错力 + 锚定感** | 主动创造一个安全空间——对方已经形成了"告诉你也没用"的习惯 |

## 四、四维组合诊断

真实场景中，情绪回应的"问题"往往是多个维度同时缺失。单一维度的问题很少见——大多数时候是一个主要问题加上一两个辅助问题。

### 组合诊断 1：对方倾诉时总给方案

**典型表现：**
> 对方说"我老板今天又在会上针对我"，你回"那你下次准备充分一点不就行了"。

**诊断：回应力缺失 + 节奏感缺失**

| 维度 | 缺失了什么 |
|------|-----------|
| **回应力 ↓** | 你没有捕捉到对方的情绪信号（委屈/愤怒/不甘），只抓取了事实信息（被老板针对） |
| **节奏感 ↓** | 你跳过了对方需要的"情绪释放阶段"，直接进入了"解决方案阶段"——太快了 |
| 容错力 ✓ | 你大概率没有反击或否认（但也不够） |
| 锚定感 ~ | 如果长期这样，对方会不再找你倾诉（锚定感也会被拖累） |

**调整方向：**
1. 先问"你现在什么感觉？"（补回应力）
2. 等对方表达了情绪再问"你需要听听我的想法，还是就想说出来？"（补节奏感）
3. 对方说"就想说说"时——闭嘴听着，给反馈（回应力训练）

---

### 组合诊断 2：对方生气时你说教

**典型表现：**
> 对方生气地说"你怎么又迟到了"，你回"你听我解释，路上堵车了，而且你上次不也迟到了吗？而且你要是提前跟我说……"

**诊断：容错力缺失 + 节奏感缺失**

| 维度 | 缺失了什么 |
|------|-----------|
| **容错力 ↓** | 对方在表达负面情绪（生气），你立刻进入"自证清白+反击"模式——你没有容纳对方的情绪 |
| **节奏感 ↓** | 对方还在情绪的顶峰，你已经开始讲道理——时机完全不对 |
| 回应力 ~ | 你听到了对方的信号，但选择了防御模式（不是听不见，是接不住） |
| 锚定感 ~ | 一次说教可能影响不大，但长期这样，对方会觉得"和你吵架没有用" |

**调整方向：**
1. 3 秒窗口：对方生气时，先深呼吸 3 秒（补容错力）
2. 先认感受，后认事实："你生气是对的，我确实迟到了"（补容错力+回应力）
3. 等对方情绪平复后再解释原因（补节奏感）

---

### 组合诊断 3：对方分享喜悦时你冷淡

**典型表现：**
> 对方说"我被选为项目负责人了！"，你回"哦，那挺好的。对了，晚上吃什么？"

**诊断：回应力缺失 + 锚定感缺失**

| 维度 | 缺失了什么 |
|------|-----------|
| **回应力 ↓** | 你没有接住对方的积极情绪信号——对方在分享喜悦，你不仅没回应，还转移了话题 |
| **锚定感 ↓** | 这种冷淡回应如果多次发生，对方会形成"跟你分享好消息也没意思"的印象——你不再是ta的第一分享对象 |
| 容错力 ~ | 不是负面情绪场景，容错力不适用 |
| 节奏感 ~ | 你转话题的时机当然不对，但根本问题是"你根本没有回应对方的情绪" |

**调整方向：**
1. 用 ACR 公式：先热情回应（"太棒了！快说说怎么选上的！"）（补回应力）
2. 把对方的喜悦变成共同的喜悦（"今晚必须庆祝！"）（补锚定感）
3. 养成一个习惯：对方主动分享任何事时——无论是好是坏——先停下手头的事，认真回应（补锚定感）

---

### 组合诊断速查

| 组合 | 问题表现 | 主要缺失 | 次要缺失 | 优先练什么 |
|------|---------|---------|---------|-----------|
| P1 | 给方案 | 回应力 | 节奏感 | FFN 刻意练习 |
| P2 | 说教 | 容错力 | 节奏感 | 3 秒窗口 |
| P3 | 冷淡喜悦 | 回应力 | 锚定感 | ACR 公式 |
| P4 | 忽冷忽热 | 锚定感 | 容错力 | 一致性练习 |
| P5 | 过度追问 | 节奏感 | 回应力 | 能量感知 |
| P6 | 翻旧账 | 容错力 | 锚定感 | 只处理当下 |
| P7 | 敷衍回应 | 回应力 | 锚定感 | 具体化+感受化 |
| P8 | 否认感受 | 容错力 | 回应力 | 3 秒窗口+FFN |

## 五、如何在其他框架中使用四维模型

四维模型是本Skill的**诊断层**——它回答"哪里不行"的问题。但它不是孤立的，它和本Skill的其他框架形成完整的闭环：

### 框架关系图

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│    🔍 四维模型（诊断） →  对方哪里不舒服？我不行在哪？    │
│         ↓                                              │
│    📖 正反案例对比法（展示）→ 别人怎么做对了？            │
│         ↓                                              │
│    🛠 ACR / FFN / FFC（改写）→ 我具体该怎么做？           │
│                                                         │
│        闭环：诊断 → 对比 → 改写                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 使用三步法

**第一步：用四维模型诊断问题**

当你的回应效果不好时，用四维模型问自己四个问题：

| 问题 | 检查哪个维度 | 如果不确定 |
|------|------------|-----------|
| 我有没有接收到对方的情绪信号？ | **回应力** | 参考速查表"总是给方案" |
| 我有没有接纳对方的情绪？ | **容错力** | 参考速查表"被触发就反击" |
| 我的节奏和对方匹配吗？ | **节奏感** | 参考速查表"冷场/追问" |
| 我的回应让对方感到安全吗？ | **锚定感** | 参考速查表"忽冷忽热" |

**第二步：用正反案例对比法找参照**

确定了是哪个维度缺失后，去参考正反案例对比：

- 四维模型中每个维度都有"正例 vs 反例"的对比表格（见上文）
- 本Skill的 `references/anti-patterns.md` 提供了六大低价值回应模式的反例库
- 对比的目的：**看清楚"好的回应长什么样"**——不是复制话术，是理解方向

**第三步：用回应框架执行改写**

有了诊断（四维模型）和参照（正反案例），最后用具体框架改写：

| 诊断结果 | 适合的改写框架 |
|---------|--------------|
| 回应力缺失 → 对方没被听到 | **FFN 倾听模型** — 先走事实→情绪→需求 |
| 容错力缺失 → 接不住负面情绪 | **消极情绪回应公式** — 认同一理解→确认 |
| 节奏感缺失 → 时机不对 | **共情回应公式** — 命名→确认→支持（先慢下来） |
| 锚定感缺失 → 对方觉得不安 | **ACR + FFC** — 稳定地回应积极和消极情绪 |

### 完整组合示例

> **场景：** 对方说"我辞职了。"
>
> **四维诊断（第一步）：**
> - 回应力 → 对方需要"被认可这个决定"，不是"被分析这个决定"
> - 容错力 → 对方可能有焦虑/不确定，需要被接住
> - 节奏感 → 先不要追问"为什么辞职"，先给空间
> - 锚定感 → 让ta知道无论怎样你都在
>
> **正反案例对比（第二步）：**
> - 反例："为什么辞职？找好下家了吗？"（直接追问事实，忽略情绪）
> - 正例参考：消极情绪回应公式中的"接受型"回应
>
> **框架改写（第三步）：**
> - 先用 FFN 确认情绪："这件事你考虑了很久吧？现在什么感觉？"（回应力）
> - 再用共情公式接住可能的负面情绪："不管是什么原因，说出来我都在，不评判你。"（容错力+锚定感）
> - 最后根据对方的回应决定下一步节奏（节奏感）

### 一句话总结

> **四维模型告诉你"哪里不行"，正反案例告诉你"什么样行"，ACR/FFN/FFC 告诉你"怎么做行"——三者组合 = 诊断 → 对比 → 改写的完整闭环。**
